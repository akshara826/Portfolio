# Akshara Mallavarapu Portfolio

A personal portfolio website built to present Akshara Mallavarapu's profile, skills, projects, internships, certifications, achievements, and contact information in a polished responsive format.

## Overview

This project is a static portfolio website designed for internships, placements, and professional visibility. It highlights academic performance, technical interests, project work, and career milestones with a modern responsive UI.

## Pages

- `index.html` - home page and profile highlights
- `about.html` - introduction, education, and timeline
- `skills.html` - technical skills and learning focus
- `projects.html` - featured project work
- `internships.html` - internship experience
- `certifications.html` - certifications and credentials
- `achievements.html` - awards and recognitions
- contact section on the home page for direct outreach

## Tech Stack

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/akshara826/Portfolio.git
```

2. Open the project folder:

```bash
cd Portfolio
```

3. Run a local server:

```bash
python -m http.server 8000
```

4. Open this in your browser:

```text
http://127.0.0.1:8000
```

You can also open `index.html` directly, but using a local server is recommended.
